# 第一题：[潮流](https://www.luogu.com.cn/problem/T752531?contestId=321591)
## 题目简述
已知$N$个$0或1的$数，问：是否满足：
1. 在$2 \leq N$的情况下，有且仅有一个数是0
2. 在$N = 1$的情况下，这个数是$1$
## 题目分析
我们只需要用程序计算有多少个0
最后用`if`特判是否满足`(N == 1 and a[1] == 1) or cnt == 1`即可
::::success[正确代码]{close}
```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n;
    cin >> n;
    int no = 0;
    for(int i = 1; i <= n; i ++)
    {
        int x;
        cin >> x;
        if(x == 0) no ++;//数0
    }
    if(n == 1 and no == 0) cout << "YES";
    else if(no == 1) cout << "YES";
    else cout << "NO";
    
}
```

# 第二题：[开关灯](https://www.luogu.com.cn/problem/T752533?contestId=321591)
## 题目简述
给定$3\times3$的一个01网格，又已知当某个数字变化的时候，上下左右的位置都会变化，例如假设原是
```textplain
A  B  C
D  E  F
G  H  I
```
如果只变化E,会变为
```textplain
 A !B  C
!D !E !F
 G !H  I
```
如果只变化C,会变为
```textplain
A !B  C
D  E !F
G  H  I
```
## 题目分析
只需要把数组开大一点，然后模拟即可
::::success[正确代码]{close}
```cpp
#include <bits/stdc++.h>
using namespace std;

const int N = 10;
int a[N][N];
bool now[N][N];

int main()
{
    for(int i = 1; i <= 3; i ++)
    {
        for(int j = 1; j <= 3; j ++)
        {
            cin >> a[i][j];
            now[i][j] = 1;
        }
    }
    for(int i = 1; i <= 3; i ++)
    {
        for(int j = 1; j <= 3; j ++)
        {
            if(a[i][j] % 2 == 1)
            {
                now[i][j] = !now[i][j];
                now[i - 1][j] = !now[i - 1][j];
                now[i][j - 1] = !now[i][j - 1];
                now[i + 1][j] = !now[i + 1][j];
                now[i][j + 1] = !now[i][j + 1];//上下左右中都更改状态
            }
        }
    }
    for(int i = 1; i <= 3; i ++)
    {
        for(int j = 1; j <= 3; j ++)
        {
            cout << now[i][j];
        }
        cout << endl;
    }
}
```

# 第三题：[卡列维奇 与 国际象棋](https://www.luogu.com.cn/problem/T702627?contestId=321591)
## 题目简述
给定一个由WB构成的$8\times8$棋盘，已知原本期盼是全W,问最少涂几笔B(任意选一行或一列)可以涂成规定棋盘的样子
## 题目分析
可以数最上面一横盘和最左边一竖列，数出有多少B,然后输出
因为有这样一组样例：
```textplain
BBBBBBBB
BBBBBBBB
BBBBBBBB
BBBBBBBB
BBBBBBBB
BBBBBBBB
BBBBBBBB
BBBBBBBB
```
实际答案是8,但数出来却是16
经过验证可得出只有这一种特殊情况
所以可以写出特判:`if(cnt == 16) cnt = 8;`
最后可写出代码:
:::success[正确代码]{close}
```cpp
#include <bits/stdc++.h>
using namespace std;

const int N = 10;
char a[N][N];

int main()
{
    int n = 8;
    for(int i = 1; i <= n; i ++)
    {
        for(int j = 1; j <= n; j ++)
        {
            cin >> a[i][j];
        }
    }
    int cnt = 0;
    for(int l = 1; l <= n; l ++)//数列
    {
        if(a[1][l] == 'B')
        {
            int f = 1;
            for(int h = 1; h <= n; h ++)//验证每一行是否都是B
            {
                if(a[h][l] == 'W') f = 0;
            }
            cnt += f;
        }
    }for(int h = 1; h <= n; h ++)//数行
    {
        if(a[h][1] == 'B')
        {
            int f = 1;
            for(int l = 1; l <= n; l ++)//验证每一列是否都是B
            {
                if(a[h][l] == 'W') f = 0;
            }
            cnt += f;
        }
    }
    if(cnt == 16) cnt = 8;
    cout << cnt;
    return 0;
}
```
