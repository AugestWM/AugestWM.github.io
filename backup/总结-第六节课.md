# 第一题：[智商测试](https://www.luogu.com.cn/problem/T754793?contestId=322993)
## 题目简述
给定一个仅包含`.`和`#`的二维数组
问能不能更改一个字符使得其包含一个只有`.`或只有`#`的一个$2\times2$的矩阵
## 题目分析
直接暴力更改每一个字符，时间复杂度较低
::::success[正确代码]{close}
```cpp
#include <bits/stdc++.h>
using namespace std;

const int N = 1e2 + 1;
char a[N][N];
int n = 4;

bool Check(char c)//函数检查是否存在一个'.'矩阵或一个'#'矩阵
{
    for(int i = 1; i <= n; i ++)
    {
        for(int j = 1; j <= n; j ++)
        {
            if(a[i][j] == c and
               a[i][j + 1] == c and
               a[i + 1][j] == c and
               a[i + 1][j + 1] == c)
                return true;
        }
    }
    return false;
}

int main()
{
    for(int i = 1; i <= n; i ++)
    {
        for(int j = 1; j <= n; j ++)
        {
            cin >> a[i][j];
        }
    }
    bool flag = 0;
    for(int i =1; i <= n; i ++)
    {
        for(int j = 1; j <= n; j ++)
        {
            char C = a[i][j];//原样存储
            a[i][j] = '#';//将这一个地方修改为'#'
            if(Check('#')) flag = 1;//检查是否有'#'矩阵，如果有，存储
            a[i][j] = C;//恢复原样
        }
    }
    for(int i =1; i <= n; i ++)
    {
        for(int j = 1; j <= n; j ++)
        {
            char C = a[i][j];//原样存储
            a[i][j] = '.';//将这一个地方更改为'.'
            if(Check('.')) flag = 1;//只要存在方案，那么就使得flag = 1
            a[i][j] = C;//恢复原样
        }
    }
    cout << (flag? "YES":"NO");
    return 0;
}
```
# 第二题：[品牌重造](https://www.luogu.com.cn/problem/T754802?contestId=322993)
## 题目简述
给定原品牌名，接下来输入字符a和b,表示将品牌重造后要将所有的a变成b，b变成a
比如
```textplain
police
o l
i e
```
会变成
```textplain
ploeci
```
## 题目分析
用一个桶数组来维护当前的字母`x`变成了什么
比如检查到`t[某一个数字] == x`然后就代表`(char)这一个数字`最后会变成x
使用这个特性，我们可以写出：
```cpp
if(a[i] == x)
{
	a[i] = y;
}
else if(a[i] == y)
{
	a[i] = x;
}
```
根据逻辑，补充出完整代码
::::success[正确代码]{close}
```cpp
#include <bits/stdc++.h>
using namespace std;

char a[250];

int main()
{
    int m, n;
    cin >> m >> n;
    string s;
    cin >> s;
    for(int i = 'a'; i <= 'z'; i ++)
    {
        a[i] = i;
    }
    while(n --)
    {
        char x, y;
        cin >> x >> y;
        for(int i = 'a'; i <= 'z'; i ++)//寻找原本i代替的
        {
            if(a[i] == x)//如果(char)i代表的是x
            {
                a[i] = y;//使得a[i]变成y
            }
            else if(a[i] == y)
            {
                a[i] = x;//同上
            }
        }
    }
    for(int i = 0; i < s.size(); i ++)
    {
        cout << a[s[i]];//输出
    }
}
```

# 第三题：[新年礼物](https://www.luogu.com.cn/problem/T754804?contestId=322993)
## 题目简述
一个人要给亲戚门塞红包
但是他没时间，只能交给机器人来做
但是机器人因为故障，只能一次投一个币
也就是说，当投币后要左右晃动一次才能再投一个币
## 题目分析
1. 特判：当前是不是0？如果是，直接走，不犹豫
2. 只要当当前这个不是0，那就继续
```cpp
while(a[i] != 0)
{
	cout << "P";
	if((a[i] - 1) > 0)
	{
		if(i == 1) cout << "RL";
		else cout << "LR";
		
	}
	a[i] --;
}
```
3. 最后只要i != n,那么让i++
::::success[正确代码]{close}
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N = 1e5 + 1;
int a[N];
    int n;

bool clear()
{
    for(int i = 1; i <= n; i ++)
    {
        if(a[i] != 0) return false;
    }
    return true;
}

int main()
{
    cin >> n;
    for(int i = 1; i <= n; i ++)
    {
        cin >> a[i];
    }
    int i =  1;
    while(!clear())
    {
        if(a[i] == 0 and i != n) 
        {
            i ++;
            cout << "R";
        }
        while(a[i] != 0)
        {
            cout << "P";
            if((a[i] - 1) > 0)
            {
                if(i == 1) cout << "RL";
                else cout << "LR";
                
            }
            a[i] --;
        }
        
        if(i != n) cout << "R";
        i ++;
    }
}
```
