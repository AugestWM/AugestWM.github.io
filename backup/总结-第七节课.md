# 第一题：[123...木头人！](https://www.luogu.com.cn/problem/T757265?contestId=324493#ide)
## 题目简述
给定一个只包含$1,2,3$的序列，问最少替换多少次可以将整个序列替换成彼此数字相等的序列
## 题目分析
分别统计将整个序列替换为$1,2,3$要多少次，取最小值
::::success[正确代码]{close}
```cpp
#include <bits/stdc++.h>
using namespace std;

const int N = 1e6 + 1;
int a[N];

int main()
{
    int n;
    cin >> n;
    for(int i=  1; i <= n; i ++)
    {
        cin >> a[i];
    }
    int A = 0, B = 0, C = 0;
    for(int i = 1; i <= n; i ++)
    {
        if(a[i] == 1 or a[i] == 2) A ++;
    }
    for(int i = 1; i <= n; i ++)
    {
        if(a[i] == 1 or a[i] == 3) B ++;
    }
    for(int i = 1; i <= n; i ++)
    {
        if(a[i] == 3 or a[i] == 2) C ++;
    }
    cout << min({A, B, C});
}
```

给定字符串，已知
-  `<`  指引导小球往`左`走

```cpp
#include <bits/stdc++.h>
using namespace std;

int n;
const int N = 2e5 + 1;
char a[N];
int sum = 0;
int t[N];

bool check(int i)
{
    while(i <= n and i >= 1)
    {
        if(a[i] == '<') i --;//模拟
        if(a[i] == '>') i ++;//模拟
        if(t[i] == 1) return false;//防止死循环
    }
    return true;//如果掉出台面，返回true
}

int main()
{
    cin >> n;
    for(int i = 1; i <= n; i ++)
    {
        cin >> a[i];
    }
    for(int i = 1; i <= n; i ++)
    {
        if(a[i] == '<' and a[i - 1] == '>'){
            t[i] = 1;
            t[i - 1] = 1;
        }//统计"><"
    }
    for(int i = 1; i <= n; i ++)
    {
        if(t[i] == 1) continue;
        if(check(i)) sum ++;//如果可以使得小球调出台面，那么sum++
    }
    cout << sum << endl;
}
```

# 第三题：[【pty2026】鸡腿](https://www.luogu.com.cn/problem/T757253?contestId=324493#ide)
## 题目描述
在一个平面直角坐标系中：
胖头鸡正在$(0,0)$，他要去往$(n,m)$
![image.png](https://free.picui.cn/free/2026/05/05/69f9e000674d6.png)
他只能上移/右移
已知，当 $x + y$ 是**偶数**时，向右移动一个单位需要消耗 $a$ 的体力，向上移动一个单位需要消耗 $b$ 的体力；当 $x + y$ 是**奇数**时，向右移动一个单位需要消耗 $c$ 的体力，向上移动一个单位需要消耗 $d$ 的体力。
问：最少多少次可以达到`(n, m)`这个点位
## 题目分析
分别计算
1. `o`：表示从0, 0到n, m要经过多少个偶数次，即$(n+m+1)\div2$
2. `j`：表示从0, 0到n, m要经过多少个奇数次，即$(n + m)\div2$
3. `ro`：表示最少多少次要右移的时候是偶数，即$min(o, n)$
4. `uo`：表示多少次上移的时候是偶数，即$o - ro$
5. `rj`：表示多少次右移的时候是奇数，即$n - ro$
6. `uj`：表示多少次上移的时候是奇数，即$j - rj$
7. `s1`：表示在确保有`ro`、`uo`、`rj`、`uj`的情况下，最少消耗多少体力，即$ro * a + uo * b + rj * c + uj * d$
8. `nuo`：表示最少多少次上移的时候是偶数
9. `nro`：表示多少次右移的时候是奇数，即$o-uo$
10. `nrj`：表示多少次右移的时候是奇数，即$n - ro$
11. `nuj`：表示多少次上移的时候是奇数
12. `s2`：表示在确保有`nro`、`nuo`、`nrj`、`nuj`的情况下，最少消耗多少体力，即$nro * a + nuo * b + nrj * c + nuj * d$
最后输出$min(s1, s2)$
::::success[正确代码]{close}
```cpp
#include <bits/stdc++.h>
#define int long long
using namespace std;

int n, m, a, b, c, d;

signed main()
{
    cin >> n >> m >> a >> b >> c >> d;
    int o = (n + m + 1) / 2,
        j = (n + m) / 2;
    int ro = min(o, n),
        uo = o - ro,
        rj = n - ro,
        uj = j - rj;
    int s1 = ro * a + uo * b + rj * c + uj * d;

    uo = min(o, m);
    ro = o - uo;
    rj = n - ro;
    uj = m - uo;
    int s2 = ro * a + uo * b + rj * c + uj * d;
    cout << min(s1, s2);
    
    return 0;
}
```
