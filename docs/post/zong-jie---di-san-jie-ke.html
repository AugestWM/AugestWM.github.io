第一题： 熊与象

给定6个数字（表示6根棍子），问：是否能搭出熊/象

  • 四根棍子代表动物的腿，这些棍子应该有相同的长度。
  • 剩下的两根棍子代表动物的头部和身体。熊的头棒必须比身棒短。 然而，大象的鼻子很长，所以它的头棒必须和身棒一样长。 请注意，腿棒与其他两种木棒之间的关系没有限制。
    熊 输出Bear
    象 输出Elephant
    不能拼出任何动物 输出Alien

题目分析

分为多种情况：

  1. 所有棒子一样max_count == 6 则为elephant
  2. 5根棒子一样max_count == 5 则为bear
  3. 最大不到4根棒子max_count < 4 则为alien
  4. 最大4根棒子else则判断是bear还是elephant
    最后完成判断
    ::::success[正确代码]{close}
#include <bits/stdc++.h>
using namespace std;

const int N = 8;
int a[N];

int main()
{
    for(int i = 1; i <= 6; i ++) cin >> a[i];
    sort(a + 1, a + 7);
    int maxx = INT_MIN, sum = 0, max_num = 0;
    for(int i = 1; i <= 6; i ++)
    {
        
        sum ++;
        if(a[i] != a[i + 1]){ 
			if(maxx <= sum)
			{
				max_num = a[i - 1];
				maxx = sum;
			}
			sum = 0;
		}
    }
    if(maxx >= 4) 
    {
    	int A = -1, B = -1;
    	int x = 0;
    	for(int i = 1; i <= 6; i ++)
    	{
    		if(x < 4 and a[i] != max_num)
    		{
    			if(A == -1) 
				{
					A = a[i];
				}
				else if(B == -1) B = a[i];
			}
		}
		if(A == B) cout << "Elephant";
		else cout << "Bear";
	}
	else
	{
		cout << "Alien";
	}
}

第二题： 尼古拉斯的画框

题目简述

告诉你有些什么棒子，问怎么组成画框，组成画框的要求是$a$边和$b$边分别有两根

题目分析

对于每一根棍子，用桶存储他在里面的数量，最后数出一共有多少对可以组成的（向下取整，int类型自动）最后再$\div\ 2$即可
::::success[正确代码]{close}

#include <bits/stdc++.h>
using namespace std;

const int N = 1e4 + 1;
int a[N];
unordered_map<int, int> mp;

int main()
{
    int n;
    cin >> n;
    for(int i = 1; i <= n; i ++)
    {
        cin >> a[i];
        mp[a[i]] ++;
    }
    int ans = 0;
    for(auto i : mp)
    {
        int s = i.second;
        ans += s / 2;
    }
    cout << ans / 2;
}

第三题： 文件夹

题目简述

在保持原顺序的情况下，怎么安排分组使得组数最少 $\text{且}$ 使得每组的负数不超过 $3$

题目分析

使用一个cnt1表示当前负数的个数
cnt2表示当前分组数
当负数(cnt1)等于3时 使得分组数(cnt2)自增1 并让负数个数 = 1
最后输出即可
::::success[正确代码]{close}

#include <bits/stdc++.h>
using namespace std;

const int N = 1e3 + 1;
int a[N];

int main()
{
    int n;
    cin >> n;
    int cnt1 = 0, cnt2 = 1;
    for(int i = 1; i <= n; i ++)
    {
        int x;
        cin >> x;
        if(x < 0) cnt1 ++;
        if(cnt1 == 3)
        {
            cnt1 = 1;
            cnt2 ++;
        }
        a[cnt2] ++;
    }
    cout << cnt2 << endl;
    for(int i = 1; i <= cnt2; i ++)
    {
        cout << a[i] << ' ';
    }
}
